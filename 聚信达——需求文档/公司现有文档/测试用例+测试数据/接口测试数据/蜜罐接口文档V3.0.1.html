1.获取蜜罐访问令牌

1.1 接口地址：

https://mi.juxinli.com/api/access_token?client_secret=XXXX&account=XXX

1.2 请求方式：

GET

1.3 输入参数：

字段 类型 备注 是否必填
account String 机构账号 是
client_secret String 机构标识码 是

1.4 正常输出:

{

  "data": {

    "access_token": "951f4e3e913240a1af709be34f279acb",

    "create_time": "1484492273000",

    "expire_in": "86400000"

  },

  "message": "获取蜜罐访问令牌成功",

  "code": "MIGUAN_ACCESS_SUCCESS",

  "spend_time": 283

}

为了数据安全的保障，数据访问令牌24小时有效

1.5 异常输出:

1.5.1 秘钥和账号不匹配

{

  "code":"MIGUAN_ACCESS_ACCOUNT_CLIENTSECRET_NOT_MATCH",

  "message":"蜜罐获取访问令牌失败，client_secret和account不匹配",

  "spend_time": 283

}

请检查输入参数

1.5.2 秘钥和账号不合法

{

  "code":"MIGUAN_ACCESS_ACCOUNT_OR_CLIENTSECRET_INVALID",

  "message":"蜜罐获取访问令牌失败，client_secret或account不合法",

  "spend_time": 283

}

检查账号和秘钥是否存在为空

1.5.3 账号已过期

{

  "code":"MIGUAN_ACCESS_ACCOUNT_EXPIRED",

  "message":"蜜罐获取访问令牌失败,account暂时不可用,{}"//{}中为账号具体情况,

  "spend_time": 283

}

检查账号和秘钥是否存在为空

1.5.4 授权接口异常

{

  "code":"MIGUAN_ACCESS_ERROR",

  "message":"蜜罐获取访问令牌内部服务异常,{}"//{}中为具体的原因,

  "spend_time": 283

}

1.6 响应数据

{

  "data": {

    "access_token": "951f4e3e913240a1af709be34f279acb", // 访问令牌

    "create_time": "1484492273000", // 创建时间

    "expire_in": "86400000" // 过期时间

  },

  "message": "获取蜜罐访问令牌成功", // 接口返回信息

  "code": "MIGUAN_ACCESS_SUCCESS", // 接口返回码

  "spend_time": 283 // 接口耗时

}

请联系对接的技术人员

2.查询蜜罐报告接口

2.1 接口地址：

https://mi.juxinli.com/api/search?client_secret=XXX&access_token=XXX&name=XXX&id_card=XXX&phone=XXX&version=v3

2.2 请求方式：

GET

2.2 输入参数：

字段 类型 备注 是否必填
name String 姓名 是
id_card String 身份证号 是
phone String 手机号 是
client_secret String 机构标识码 是
access_token String 安全凭证码 是
version String 版本号 是
origin_company String 平台名称 否
product_type String 产品类型 否

2.3 正常输出：

{

  "code":"MIGUAN_SEARCH_SUCCESS",

  "message":"获取蜜罐查询成功",

  "data":{         //响应数据

  }, 

  "spend_time": 283

}

特别提示： data 下的数据字典将做增量更新，请兼容此种模式
示例：{"k1":"v1","k2":"v2"} 可能会升级成 {"k1":"v1","k2":"v2","k3":"v3"}

2.4 异常输出：

2.4.1 未开通访问权限

{

  "code":"MIGUAN_SEARCH_PERMISSION_DENY",

  "message":"蜜罐查询权限未开通",

  "spend_time": 283

}

请联系商务人员开通权限

2.4.2 账号已过期

{

  "code":"MIGUAN_SEARCH_ACCOUNT_EXPIRED",

  "message":"蜜罐查询账号暂时不可用,{}", //{}为账号具体情况

  "spend_time": 283

}

联系商务人员或技术人员处理是否存在欠费情况

2.4.3 访问令牌或秘钥不合法

{

  "code":"MIGUAN_SEARCH_ACCESSTOKEN_OR_CLIENTSECRET_INVALID",

  "message":"蜜罐查询访问令牌或秘钥不合法",

  "spend_time": 283

}

核对 “访问令牌” 和 “账号秘钥”是否存在为空

2.4.4 访问令牌和秘钥不匹配

{

  "code":"MIGUAN_SEARCH_ACCESSTOKEN_CLIENTSECRET_NOT_MATCH",

  "message":"蜜罐查询访问令牌和秘钥不匹配",

  "spend_time": 283

}

核对 “访问令牌” 和 “账号秘钥”是否匹配

2.4.5 访问令牌已过期

{

  "code":"MIGUAN_SEARCH_ACCESSTOKEN_EXPIRED",

  "message":"蜜罐查询访问令牌已经过期",

  "spend_time": 283

}

重新获取访问令牌

2.4.6 身份证不合法

{

  "code":"MIGUAN_SEARCH_IDCARD_INVALID",

  "message":"蜜罐查询身份证不合法",

  "spend_time": 283

}

检查身份证是否为空或是否符合国标算法

2.4.7 姓名不合法

{

  "code":"MIGUAN_SEARCH_NAME_INVALID",

  "message":"蜜罐查询姓名不合法",

  "spend_time": 283

}

检查姓名是否为空或包含特殊符号（只允许存在·）

2.4.8 手机号码不合法

{

  "code":"MIGUAN_SEARCH_PHONE_INVALID",

  "message":"蜜罐查询手机号码不合法",

  "spend_time": 283

}

检查手机号码是否为空或格式不正确

2.4.9 作为渠道商终端查询公司名称不合法(特殊)

{

  "code":"MIGUAN_SEARCH_ORIGIN_COMPANY_INVALID",

  "message":"蜜罐查询终端查询公司名称不合法",

  "spend_time": 283

}

传入正确的终端公司名称

2.4.10 内部服务异常

{

  "code":"MIGUAN_SEARCH_ERROR",

  "message":"蜜罐查询内部服务异常,{}", //{}中为错误细节

  "spend_time": 283

}

联系技术对接人员

2.4.11 达到当日查询数量上限

{

  "code":"MIGUAN_SEARCH_OVERLOAD_QUERY_LIMIT",

  "message":"蜜罐查询达到当日查询数量上限,{}", //{}是具体限制数量

  "spend_time": 283

}

蜜罐查询达到当日查询数量上限，咨询商务询问上限数量

2.4.12 疑似重复查询异常

{

  "code":"MIGUAN_SEARCH_SIMILAR_QUERY_ERROR",

  "message":"蜜罐查询疑似重复查询异常",

  "spend_time": 283

}

同一机构账号下，查询蜜罐用户的姓名、手机号和身份证号三要素两两组合，一天内查询次数不超过四次

2.4.13 缺失产品查询权限

{

  "code":"MIGUAN_SEARCH_MISSING_PRODUCT_QUERY_AUTHORITY",

  "message":"缺失产品查询权限",

  "spend_time": 283

}

联系商务开通相关权限
